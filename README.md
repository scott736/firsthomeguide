# FirstHomeGuide.ca

Canada's most comprehensive free resource for first-time home buyers. An 8-module guide covering everything from financial readiness to life after closing.

## Tech Stack

- Astro + Starlight
- React + Tailwind CSS
- Vercel (hosting & serverless)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) to view the site locally.

## Deployment

Deployed automatically via Vercel on push to `main`.
